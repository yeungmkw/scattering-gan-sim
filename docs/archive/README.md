# Documentation Archive

This directory preserves superseded material without keeping it in the active
documentation path.

- [`legacy/`](legacy/) contains public results from the pre-Luo-2022 prototype.
  They remain useful for tracing the project's development, but they are not
  current scientific baselines.
- `internal/` contains local-only reproduction contracts, evidence ledgers,
  diagnostic notes, and literature surveys. The directory is intentionally
  excluded from Git.

Archived documents may be cited for historical context or reused for data
provenance. Current claims and project decisions must come from the documents
in the parent `docs/` directory.
